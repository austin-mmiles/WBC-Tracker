1

Install Python (if you don't have it)

Download from python.org — it's free. Then install requests once: pip install requests

2

Run the fetcher script

In your terminal/command prompt, navigate to the folder with fetch_wbc_stats.py and run: python3 fetch_wbc_stats.py — it will create a fresh wbc_dashboard.html with current scores.

3

Share the HTML file

Send the generated wbc_dashboard.html via iMessage, email, or AirDrop to your group. Anyone can open it in a browser — no internet required to view it.

4

Automate it (optional)

On Mac/Linux, add a cron job to run every 30 minutes during games: open Terminal, type crontab -e, and add: */30 * * * * python3 /path/to/fetch_wbc_stats.py. On Windows, use Task Scheduler.

5

Host it live (optional)

For a shared live URL, upload wbc_dashboard.html to GitHub Pages (free). Run the script locally, push the file to your repo, and GitHub Pages serves it. The URL stays the same each time you push an update.
