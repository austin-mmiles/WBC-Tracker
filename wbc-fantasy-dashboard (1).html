Open this file in any browser

Just double-click wbc-fantasy-dashboard.html. No server, no install, no login needed. Works on phone or desktop.

2

Share with your group via GitHub Pages (free)

Create a free GitHub account → New repo → Upload this file as index.html → Go to Settings → Pages → Enable from main branch. You get a live URL like yourusername.github.io/wbc-fantasy that anyone can open on their phone.

3

Stats update automatically

The page polls MLB's free Stats API every 60 seconds. During WBC games, data is typically 2–5 minutes behind live action. The tournament ID is pre-configured — no changes needed.

4

If a player isn't found

MLB's API sometimes uses slightly different name spellings. Open the file in a text editor and find the ROSTERS section. You can add a player's MLB ID directly next to their name entry for exact matching. Player IDs can be found at mlb.com/player/[name].

5

CORS issue? Use a proxy

If running locally you may see a CORS error. Either host on GitHub Pages (solves it), or install the Allow CORS browser extension temporarily. The GitHub Pages approach is the cleanest solution.
